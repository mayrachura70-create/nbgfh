## 📱 Descripción

**Wiñay Tika** es una aplicación web moderna y juvenil para vender rosas eternas elaboradas a mano con cintas de la más alta calidad. El nombre "Wiñay Tika" significa "Flor Eterna" en quechua, reflejando la esencia del producto.

La aplicación combina:
- ✨ **Diseño moderno y atractivo** con gradientes y animaciones fluidas
- 🛍️ **Carrito de compras funcional** con persistencia en localStorage
- 📦 **Catálogo de productos** con filtros por categoría
- 💬 **Sistema de contacto** integrado con WhatsApp
- 📱 **Diseño responsivo** para todos los dispositivos

## 🚀 Cómo Usar

### 1. **Ver Productos**
- Desplázate a la sección "Productos"
- Usa los filtros para buscar por categoría
- Haz clic en "Ver Detalles" para más información

### 2. **Comprar**
- Abre el detalle del producto
- Ajusta la cantidad deseada
- Haz clic en "Agregar al Carrito"
- Accede al carrito desde el botón superior
- Haz clic en "Comprar" para finalizar en WhatsApp

### 3. **Contactar**
- Ve a la sección "Contacto"
- Completa el formulario
- Serás redirigido a WhatsApp automáticamente

## 🔐 Datos Persistentes

El carrito se guarda automáticamente en el navegador usando `localStorage`, permitiendo:
- Guardar compras sin estar registrado
- Recuperar el carrito al volver a visitar
- Sincronización instantánea entre pestañas

## 💡 Personalización

### Cambiar Número de WhatsApp
En `script.js`, busca y actualiza:
```javascript
const enlaceWhatsApp = `https://wa.me/573214567890?text=...`
```

### Agregar Productos
En el array `productos` de `script.js`:
```javascript
{
    id: 10,
    nombre: "Tu Producto",
    categoria: "tu-categoria",
    precio: 50000,
    descripcion: "Descripción",
    emoji: "🌹"
}
```

### Cambiar Colores
En `styles.css`, modifica las variables CSS:
```css
:root {
    --primary-color: #FF1493;
    --secondary-color: #FF69B4;
    /* ... */
}
```
